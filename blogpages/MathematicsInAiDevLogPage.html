Learning Journal | Ai | Mathematics In Ai

1st Sept, 2022

Vectors

A point is a location in space. Movement in this space can be defined by a vector which is a direction and length (magnitude), and has no starting point.

By adding a vector to the position of a game character the character will be moved to a new position in space.

We can perform this by using Unity's Translate method. E.g.

                transform.Translate(6f, 0, 2f);

Adding this same vector to another game character will move it by the same amount, but they will not end up in the same location as they most likely have different starting points to each other.


Moving a Character Towards a Target

To make a character go towards a target location we need to calculate the vector between the character and it's target.

We can find this vector by taking the character's position away from the target position.

                vector = target.position - character.position;

However we cannot add this entire vector in one go as this will result in the character just teleporting to it's target location. We need to be able to add a shorter vector that points in the same direction after each update.


This can easily be achieved by dividing the vector. To get a vector that is half the size you would divide it by two. This would take the character two updates to reach it's goal.

If we divide a vector by a fixed number each time then the characters speed will change depending on the distance to it's target. Even if divided by the same number, the character will move much faster the longer the distance that needs to be travelled.

Rather than dividing a vector by the same amount, we want to reduce the smaller vectors to a standard size such that they are all the same length. This standard size is one. This can be found by dividing the larger vector by it's magnitude. This is called normalisation. This will result in the character now moving at a constant speed regardless of the distance that is needed to be travelled.

                direction = target.position - character.position;
direction.magnitude   // length of the vector
direction.normalised // vector of length 1

Calculating The Distance Between Two Points

Using Pythagoras Theorem the distance between two points can be calcuated.

                float distance = Mathf.Sqrt(Mathf.Pow(zombie.transform.position.x - transform.position.x, 2) +
                            Mathf.Pow(zombie.transform.position.y - transform.position.y, 2) +
                            Mathf.Pow(zombie.transform.position.z - transform.position.z, 2));

The above can also be achieved more simply by using Unity's built-in methods.

                float distance = Vector3.Distance(zombie.transform.position, transform.position);

A third option for achieving this is to calculate the vector between two points which we can then use to calculate the length of the vector (the length of the vector being the distance between two points).

                Vector3 zombieToPlayer = (transform.position - zombie.transform.position).magnitude;

Calculating The Dot Product

The Dot Product is the sum of the multiplication of coordinate values in two vectors.

In the example below multiplying the x values 3 and 6, the y values 3 and -1, and then adding the result gives us the dot product.

i.e. v.x * w.x + v.y * w.y

i.e. 3 * 6 + 3 * -1 = 15

                v = (3,3)
w = (6,-1)
  • If the dot product is equal 0 then the angle is exactly 90 degrees.
  • If the dot product is greater than 0 then the angle is less than 90 degrees.
  • If the dot product is less than 0 then the angle is greater than 90 degrees.

Using the Dot Product allows us to determine where one game object is in relation to another. For example you can determine whether a game object is in front or behind something. If a game object is in front of a game object then the Dot product will be greater than 0. If a game object is behind another game object then the Dot product will be less than zero.


To calculate the angle that a tank would need to turn towards some fuel the forward facing vector of the tank is needed as well as the vector between the tank and the fuel. Getting the forward facing vector is easy in Unity. In a 2D environment the forward vector is stored in the tanks Up vector. In a 3D environment the tanks Forward vector would be used.

                Vector3 tankForward = tank.transform.up;
Vector3 fuelDirection = fuel.transform.position - tank.transform.position;

// Debug
Debug.DrawRay(tank.transform.position, tankForward, Color.green, 2);
Debug.DrawRay(tank.transform.position, fuelDirection, Color.red, 2);

Calculating The Cross Product

The cross product of two vectors results in another vector. It is perfomred on vectors in three dimensions.

If the cross products z value is less than zero or negative then a clockwise turn should be performed. If the cross products z value is more than zero then a anticlockwise turn should be performed.
